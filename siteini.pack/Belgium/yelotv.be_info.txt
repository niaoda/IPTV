   Generating the cookie file
______________________________________________________________________________

1. open a browser and go to http://yelotv.be/tvgids
2. click on the button "Mijn zenders"
3. in the next window, click "Toon alle zenders"
4. be sure you can now see the EPG schedule of all the channels!
5. save the cookie file next to yelotv.be.ini and name it yelotv.be_cookies.txt
( see http://webgrabplus.com/node/186 for browser specific howto's )
